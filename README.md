# 物理實驗演示網站 / Physics Experiment Demo Website

## 項目簡介 / Project Introduction

這個網站用於展示各種物理實驗的互動式演示。通過直觀的可視化和模擬，幫助學生和教育工作者更好地理解物理概念和原理。

This website showcases interactive demonstrations of various physics experiments. Through intuitive visualization and simulation, it helps students and educators better understand physics concepts and principles.

## 功能特點 / Features

- 互動式物理實驗模擬 / Interactive physics experiment simulation
- 即時參數調整 / Real-time parameter adjustment
- 可視化物理現象 / Visualization of physical phenomena
- 適用於教育和學習環境 / Suitable for educational and learning environments
- 響應式設計，支持多種設備 / Responsive design, supports multiple devices

## 實驗演示內容 / Experiment Demonstrations

### 1. 力學實驗 / Mechanics Experiments
- 牛頓運動定律演示 / Newton's Laws of Motion demonstration
- 能量守恆定律模擬 / Conservation of Energy simulation
- 彈簧振子實驗 / Spring oscillator experiment
- 自由落體運動 / Free fall motion

### 2. 電磁學實驗 / Electromagnetism Experiments
- 歐姆定律驗證 / Ohm's Law verification
- 電路連接模擬 / Circuit connection simulation
- 磁場線可視化 / Magnetic field line visualization
- 電磁感應演示 / Electromagnetic induction demonstration

### 3. 波動與光學實驗 / Waves and Optics Experiments
- 波的干涉和衍射 / Wave interference and diffraction
- 光的折射和反射 / Light refraction and reflection
- 雙縫干涉實驗 / Double-slit interference experiment
- 駐波現象 / Standing wave phenomenon

## 技術棧 / Technology Stack

- **前端 / Frontend**: HTML5, CSS3, JavaScript
- **可視化庫 / Visualization Library**: Canvas, SVG
- **物理引擎 / Physics Engine**: 自定義物理模擬算法 / Custom physics simulation algorithms
- **響應式設計 / Responsive Design**: CSS Media Queries
- **部署 / Deployment**: GitHub Pages

## 如何使用 / How to Use

1. 訪問網站首頁 / Visit the website homepage
2. 選擇感興趣的物理實驗類別 / Choose the physics experiment category of interest
3. 點擊具體實驗項目 / Click on a specific experiment
4. 調整參數觀察實驗變化 / Adjust parameters to observe changes
5. 查看實驗說明和理論背景 / View experiment description and theoretical background

## 安裝與運行 / Installation and Running

```bash
# 克隆項目 / Clone the project
git clone https://github.com/paulng99/paulng99.github.io.git

# 進入項目目錄 / Enter the project directory
cd paulng99.github.io

# 在瀏覽器中打開 index.html / Open index.html in browser
open index.html
```

或者直接訪問在線版本 / Or visit the online version directly: [https://paulng99.github.io](https://paulng99.github.io)
